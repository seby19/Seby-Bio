Simple Neural Net

This is and educational project developed to understand the inner workings of a neural network. The library constructs a three layered neural network and allows you to specify the number of nodes in each layer. It then takes a list of values for inputs and outputs , and trains the the neural net using backpropogation.

It is basically a library through which you can leverage the function of a neural network. The library is written in Javascript.
